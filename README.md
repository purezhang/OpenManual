# OpenManual

OpenManual 是一个开源的说明书存档项目，致力于收集、整理并永久保存各类生活家电与数码外设的说明书。

由于纸质说明书易丢失、厂商官网老旧文档下载不便，本项目通过社区协作，建立一个结构清晰、纯净无广告的数字资源库。

## 📂 目录结构规范

项目采用扁平化的分类结构，按英文首字母排序，不设数字前缀以便于扩展：

`[分类] / [品牌] / [型号] / [文档内容]`

- **分类**：按设备场景划分（如 IT_Peripherals, Kitchen_Dining）。
- **品牌**：使用品牌官方英文名称（如 Razer, Midea, Logitech）。
- **型号**：设备具体的型号目录。
- **文档内容**：
  - `PDF/`：存储原始说明书。
  - `PICs/`：存储快速上手指南图片、接线图、背板接口图等。
  - `README.md`：核心参数摘要（如：键盘快捷键列表、咖啡机故障代码对照表）。

## 🗂️ 分类指南

- **Cleaning_Appliances**：清洁家电（扫地机、洗地机）。
- **Climate_Control**：环境电器（空调、净化器、除湿机）。
- **Computing_Network**：计算与网络（路由器、交换机、DDI 设备）。
- **IT_Peripherals**：IT 外设（键盘、鼠标、显示器、数码周边）。
- **Kitchen_Dining**：厨房与饮食（咖啡机、饮水机、烹饪电器）。
- **Personal_Care**：个人护理（剃须刀、吹风机）。
- **Smart_Home_Security**：智能家居与安防（摄像头、智能门锁）。

## 📝 命名约定

1. **多语言处理**：
   文件名应包含语言后缀，例如：
   - `User_Manual_ZH.pdf` (中文说明书)
   - `User_Manual_EN.pdf` (英文说明书)
2. **文件名**：尽量使用下划线 `_` 代替空格，保持跨平台兼容性。

## 🚀 如何贡献

1. Fork 本仓库。
2. 按照上述规范创建对应的文件夹。
3. 上传说明书（优先选择官网电子版）。
4. 在型号目录的 README.md 中提取最实用的“快速配置”或“快捷键”信息。
5. 提交 Pull Request。
